# Invoice System

## Project Overview

Invoice System is a web-based application developed to manage and print customer invoices for home services such as:

- Plumber services
- Gas cylinder delivery
- Supermarket order delivery
- General home maintenance services

The system allows users to manage customer information, create invoices, calculate totals, and print invoices in a professional format.

## Features

### Customer Management
- Add new customers
- Edit customer information
- Delete customers
- View customer details

### Invoice Management
- Create customer invoices
- Add multiple invoice items/services
- Calculate subtotal, tax, and total amount
- Store invoice history

### Invoice Printing
- Generate printable invoices
- Display customer and invoice details
- Print invoices directly from the application
